Assignment 3: Input Output!

Overview

I created a a schematic, circuit, and code that uses a transistor to control load power separate from logic power.

Schematic and Circuit

For my circuit, I used the LED strip, N channel MOFET (transistor) and the IR sensor .

The actual circuit is shown below. The gate (G) of transistor is connected to pin 11, drain (D) to the LED strip and an external power source, and the source (S) to ground. In addition, the Y channel of the receiver is connected to pin 6, R channel to the 5 volt pin on the arduino board and G channel to ground.

Code Snippet

The hex code 0xFF906F represents the up arrow button on the remote while the hex code 0xFFE01F represents the down arrow.
